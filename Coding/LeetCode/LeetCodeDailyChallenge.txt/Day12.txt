Question: You are given an array of positive integers nums and want to erase a subarray containing unique elements. The score you get by erasing the subarray is equal to the sum of its elements.

Return the maximum score you can get by erasing exactly one subarray.

An array b is called to be a subarray of a if it forms a contiguous subsequence of a, that is, if it is equal to a[l],a[l+1],...,a[r] for some (l,r).

Approach: https://leetcode.com/problems/maximum-erasure-value/discuss/2140577/An-Interesting-Optimisation-or-JAVA-Explanation




Solution: 1
class Solution {
    public int maximumUniqueSubarray(int[] nums) {
        int[] arr = new int[10001];
        int start = 0;
        int end = 1;
        int maxValue = nums[start];
        int sum = nums[start];
        arr[nums[start]]=1;
        while(end<nums.length){
            if(arr[nums[end]]==1){
                maxValue = Math.max(sum,maxValue);
               while(end<nums.length && start<nums.length && (nums[start]!=nums[end])){
                   arr[nums[start]]=0;
                   sum-=nums[start++];
               }
                arr[nums[start]]=0;
                sum-=nums[start++];
            }else{
                arr[nums[end]]=1;
                sum+=nums[end++];
            }
        }
        return Math.max(sum,maxValue);
    }
}
